** README **
PROYECTO DESARROLLO DE MODELO FORMATIVO AWAKELAB

Problema:
Desarrollo de sistema de evaluación de cumplimiento de aprendizaje y desarrollo por competencia como uno de sus procesos internos.

Solución:
Se procede a la construcción de base de datos en Sql Developer, con el fin de ser el mantenedor de la información de los alumnos y sus evaluaciones a lo largo de su proceso formativo.

Este proyecto contendrá lo siguiente, el cual es parte del proceso de pruebas de este nuevo sistema:
- Modelo lógico en OracleDataModeler.
- Modelo relacional en OracleDataModeler.
- DDL generado por OracleDataModeler.
- Script SQL que registre al menos 2 evaluaciones, con 10 preguntas y 4 alternativas.
- Script SQL que registre al menos 2 cursos de 10 estudiantes que respondan estas
evaluaciones.
- Script SQL que responda las preguntas enumeradas anteriormente.
